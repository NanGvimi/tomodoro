关于 PIP 模式

Tomodoro 支持 “始终在顶部 ”模式，即 “画中画 ”模式，可直接在浏览器中使用，无需安装任何电子 app。
基于 Chromium 的浏览器支持 PIP 模式（已在 Chrome 和 Edge 上测试）。
Safari 浏览器尚未测试，但可能可以使用。
在桌面版火狐浏览器上，PIP 按钮只会让视频可见，但不会激活 PIP。要切换到 PIP 模式，请右键单击视频并选择 “在画中画模式下观看”。


安卓版 Chrome 浏览器上的 PIP 模式

如果您使用的是 Android 版 Chrome 浏览器，那么 PIP 按钮将使视频元素可见。要切换到 PIP 模式，请将视频设为全屏，然后转到设备的主屏幕（按主屏幕按钮或使用导航手势）而不退出全屏。PIP 模式将被激活。

Tomodoro

Tomodoro 是一款开源 Pomodoro 网络应用程序，具有 PIP 模式、白噪声、任务、统计和离线支持等超酷功能。


新功能

  • 任务与统计

    你现在可以创建任务，并跟踪专注于这些任务所花费的时间。所有数据都存储在浏览器中。

  • 白噪音

    现在，您可以在一轮专注过程中聆听白噪音。

Safari 浏览器尚未测试，但可能可以使用。
